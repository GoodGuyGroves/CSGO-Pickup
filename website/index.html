#gopugs

Bot Commands

!sg - Starts a game

!add - Randomly add to either team A or B

!add [a|b] - Add to either team A or B

!rem - Removes you from the game

!status - Displays the players added and spots left

The Process

IRC
After a game has been started with the !sg command, the first 10 people to type !add (or !add [a|b]) will get a spot in the game. After all 10 spots have been filled, the bot will wait 60 seconds for any changes to be made such as player removals or swaps. After the timer the bot will PM every added player with the server IP address and password.

CS:GO Server
Now that you have the server details from the bot, open CS:GO and connect. The server has a plugin running called warmod that will handle the in game aspects of the pickup. Once again you wait for all 10 people to join and type /ready in chat, the plugin will restart the round and go live with the competitive server settings (ESL config).
